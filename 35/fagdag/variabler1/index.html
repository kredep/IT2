Variabler

Når man arbeider med JS må man deklarere variabler med et navn.

Etter man har deklarert en variable må man tilordre variabelen en verdi

Deklarering og tilordning av variabler kan se slik ut:

var myString = "En tekst"; // Variabeltypen 'string', tekst

var myInt = 10; // Variabeltypen 'integer', heltall

var myNum = 3.14; // Variabeltypen 'number' (float), desimaltall
